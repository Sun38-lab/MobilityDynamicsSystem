## プロジェクト概要 (Overview)
**PC連動型 1軸姿勢制御（ジンバル）システム**


本プロジェクトは、STM32マイコンによるリアルタイム姿勢制御と、C#で構築したPC側の可視化・チューニングツールを一気通貫で連携させたシステムです。<br>
通常のPID制御（位置制御）では、安価なサーボモーター特有の静止摩擦やギアのガタつきにより、目標値付近で激しい発振（ハンチング）が発生します。本システムでは、PID出力を「絶対位置」ではなく「角速度（スピード）」として扱う制御モデルを採用し、偏差の大きさに応じて動的に制御ロジックを切り替えることで、産業用機器に迫る安定した着地を実現しています。<br>
また、組み込み領域において「ブラックボックス化しやすい物理現象と制御状態」を、自作のPCツールを用いて波形として可視化することで、理論的かつ効率的なシステム開発・デバッグ環境を構築しました。

## コア技術：ハイブリッド姿勢制御アルゴリズム

単一の制御理論では対応できない領域を、4つのフェーズに分けたハイブリッド制御で解決しています。

### 1. 大ズレ時のバンバン制御（偏差 ±10度以上）
外乱等で姿勢が大きく崩れた際は、PIDの計算をキャンセルし、モータを最大速度（PWM変化量 ±3000.0）で一気に目標方向へ引き戻します。これにより、応答遅れを最小限に抑えます。

### 2. 接近フェーズ（偏差 ±4度〜10度）
摩擦の影響が少ないこの領域では、やや強めの比例ゲイン（Kp）を掛け、一気に目標付近まで寄せていきます。

### 3. D制御による着地安定化とソフトバウンダリ（不感帯突入時：±2.0度）
目標付近での着地ショックとハンチングを防ぐための最重要フェーズです。
*   **ソフトバウンダリの導入:** 不感帯（±2.0度）を「はみ出した分」だけを有効な誤差（`active_error`）として抽出し、Kpを掛けます。
    $$v = Kp \times e_{active} + Kd \times \dot{\theta}$$
*   **ノイズの足切りとブレーキ:** ジャイロセンサーの角速度ノイズ（5.0dps以下）を足切りしつつ、機体が動いている間は微分ゲイン（Kd）によるブレーキを優先的に効かせ、発振（プルプル現象）を完全に封じ込めます。

### 4. I制御による定常偏差の解消（吸い付き着地）
不感帯の中で完全に静止した（角速度ノイズ以下になった）ことを検知すると、超微小な積分ゲイン（Ki）による力を加えます。これにより、静止摩擦をジワジワと突破し、最終的に0度へ完全に「吸い付く」ような着地を実現します。

## システムアーキテクチャ

<!-- ここにシステム全体のブロック図（Draw.ioやMermaidなどで作成）を配置 -->

### 組み込み側 (STM32 / C言語 / FreeRTOS)
*   **高精度なタスクスケジューリング:** `osDelayUntil`を用いて、センサー読み取りタスク（優先度：高）を10ms周期で正確に実行しています。取得したデータはFreeRTOSのメッセージキューを介してUART送信・制御タスクへ安全に渡されます。
*   **相補フィルターによる姿勢角推定:** 加速度センサー（長期的な安定性）とジャイロセンサー（短期的な応答性）の利点を組み合わせ、以下の相補フィルター式でノイズの少ないピッチ角・ロール角を算出しています。
    $$\theta_{comp} = \alpha (\theta_{comp} + \dot{\theta}_{gyro} \Delta t) + (1-\alpha) \theta_{acc}$$
*   **ハードウェア・フェールセーフ:** I2Cバスのハングアップ対策として、初期化時にSCL/SDAピンをGPIO出力として手動でトグルさせ、STOPコンディションを強制発行してバスを復旧させる泥臭い実用コードを実装しています。

### PC側 (C# Windows Forms / ScottPlot)
*   **リアルタイムモニタリング:** 受信したシリアルデータから、補正後のピッチ角やジャイロ角速度を50ms間隔でScottPlotを用いてリアルタイムに波形描画します。
*   **オンザフライでのパラメータチューニング:** 再コンパイルすることなく、GUI上からUART経由で$Kp$、$Ki$、$Kd$の各ゲインや目標角度を動的に変更可能です。
*   **緊急停止機能:** `STOP`コマンドを送信することで、STM32側の状態遷移をエラー状態へ移行させ、即座にPWM出力を停止する安全機能を備えています。

## 開発環境とセットアップ

*   **IDE:** STM32CubeIDE, Visual Studio Code, Visual Studio 2022
*   **バージョン管理:** Git / TortoiseGit

### ビルドと実行手順
1. 本リポジトリをクローンします。
2. `Firmware` フォルダ内のプロジェクトを STM32CubeIDE で開き、ビルド・書き込みを行います。
3. 機体を水平な場所に置き、リセットボタンを押します。約1秒間のジャイロキャリブレーション（オフセット計算）が自動で実行されます。
4. `Dashboard` フォルダのC#ソリューションをビルドし、アプリを立ち上げます。正しいCOMポートを指定するとリアルタイム描画が開始されます。
